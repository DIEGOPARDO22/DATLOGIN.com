HTML

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra creado originalmente para documentos cientificos, un gracias al creador de esto, amen

en cambio lenguajes de otras tecnologias como CSS, JavaScript, complementa a nuestra pagina web haciendola mas completa, pero esto requiere de una cierta habilidad
